# Pricey
Pricey e' un comparatore di prezzi realizzato ad hoc.
L'algoritmo non e' molto robusto dato che la struttura dei file excel che vengono letti potrebbe cambiare nel tempo.

# Installazione ed Esecuzione
`mvn clean install `
Il jar si trovera' nella cartella target assieme alle dipendenze (la cartella libs)
